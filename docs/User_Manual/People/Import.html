User Manual People Import

You can import contacts if you select the "Import people..." menu item in the "Export" dropdown menu.

A page will open with a Excel-like grid. You can upload and select an Excel file to load in this grid. The first sheet of the Excel file will be loaded, after which you can make changes before clicking "Import".

Unique email addresses

Email addresses are unique in the system. If the email column of a row contains an email address that already exists, the row will be skipped.

Required fields

If one or more required fields are not filled in for a row, this row will be skipped:

  • Full name
  • Email address

User role

The user role can be either a number, or a name:

  • 1 or Admin
  • 2 or Manager
  • 3 or Employee
  • 4 or Contractor
  • 5 or Client
  • 6 or Agent
  • 7 or Lead

If the role column is empty or can't be matched with an existing role, it is set to Employee by default.

Country

A country must be the two-letter country code according to the ISO 3166-1 alpha-2 codes.
